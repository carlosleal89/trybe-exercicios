#comandosGit.txt
1-Criar repositório no GitHub
2-Clonar repositório para PC
3-Criar uma nova branch para começar as alterações.
4-Criar uma pasta local para receber as novas pasta de exercicios
4-Faz as alterações/exercicios
5-Git add, Git commit
6-Git push -u origin "nome da branch" 
7-Vai no GitHub e faz um pull/request
9-VsCode, volta pra branch main
10-git pull origin main
